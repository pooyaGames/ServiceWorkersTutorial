# Comperhensive Service workers tutorial

In modern web, service workers are playing a huge role. but still there are not so many comprehensive resources to learn about them.
I will try to add examples about how to use service workers in different cases in this repository.

# When do service workers run ?

Service workers run based on events.

this means that when certain things happen then service worker file starts to execute, things like fetching a file from your server or receiving a message from server like notification or when your website is added to user's device as an app.
